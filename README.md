T7\_nblA
========

This repository contains the [snakemake](https://snakemake.readthedocs.io/) workflow used for the bioinformatic analyses in Nadel et al [Viral NblA proteins negatively affect oceanic cyanobacterial photosynthesis](https://www.nature.com/articles/s41586-025-09656-x) _Nature_ 648, 434-442 (2025)
